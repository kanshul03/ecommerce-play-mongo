## E-commerce Rest API using Java Play framework and Mongodb.

use following path to check out inventory/all the products - 

* GET             /product/
* GET             /product/:id
* POST            / product/
* PUT             /product/
* DELETE          /product/:id

---

use following paths for cart -
* POST            /cart/
* GET             /cart/:userId
* PUT             /cart/:userId/
* DELETE          /cart/:userId/
